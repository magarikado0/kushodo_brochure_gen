# 京大書道部パンフレット生成ツール

作品情報フォームから、パンフレットの作品一覧と作品紹介ページを自動生成します。

## 使い方

1. `input` フォルダに `作品情報フォーム.xlsx` を入れます。
2. `templates` フォルダに前回パンフレットの `冬樟展パンフ.docx` を入れます。
3. 実行ファイルをダブルクリックします。
  - Windows: `run.bat`
  - Mac: `run.command`
4. `output` フォルダに生成された Word ファイルができます。

初回だけ、環境準備のためにインターネット接続が必要です。

基本的に触るのは `input`、`templates`、`output`、`run.bat` / `run.command` だけです。その他のファイルやフォルダはそのままにしてください。

Mac で `run.command` が開けない場合は、右クリックして「開く」を選んでください。それでも開けない場合は、詳しい人に聞いてください。

## 作品情報フォームの要件

フォームには、次の2シートが必要です。

- `個人`
- `合作`

列の順番は変わっても大丈夫です。ただし、列名の先頭部分は変えないでください。

個人シートに必要な列:

- `氏名`
- `ふりがな`
- `学年`
- `臨書 or 創作`
- `書体`
- `作品名`
- `作品の向き`
- `作品サイズ`
- `展示場所`
- `表装形式`
- `釈文`
- `作品コメント`

合作シートに必要な列:

- `合作参加者全員分`
- `臨書 or 創作`
- `書体`
- `作品名`
- `作品の向き`
- `作品サイズ`
- `展示場所`
- `表装形式`
- `釈文`
- `作品コメント`

学年は `2回生`、`二回生`、`B2`、`修士1回生`、`M1` などに対応しています。出力時は漢数字に変換されます。

合作参加者は、名前の後ろに学年を付けてください。

```text
加藤 杏次郎（B4） 星野 真帆（B4）
```

## テンプレートファイルの要件

`templates/冬樟展パンフ.docx` は、前回のパンフレットをそのまま使う想定です。

テンプレートには、次の内容が必要です。

- `作品一覧` という見出し
- `個人作品` という見出し
- 学年見出しの例として `一回生`
- 作品紹介ページの開始位置として `賛助作品`
- 作品紹介ページ内の、番号入りテキストボックス
- 後ろの固定ページの開始位置として `【顧問・部員紹介】`

完全に新しい Word ファイルではなく、前回パンフレットをコピーして使ってください。
